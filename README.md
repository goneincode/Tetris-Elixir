# Tetris-Elixir

A terminal-based Tetris game written in Elixir.

## Features
- Full Tetris game logic (movement, rotation, collision)
- Line clearing and scoring
- Next level progression (score based)
- Raw mode terminal input (instant response)
- Unicode/ASCII graphics

## How to Play

**Prerequisites**: You need Elixir installed.

1.  Clone the repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Tetris-Elixir.git
    cd Tetris-Elixir
    ```

2.  Run the game:
    ```bash
    mix run -e "Tetris.play"
    ```

## Controls

- **a**: Move Left
- **d**: Move Right
- **w**: Rotate
- **s**: Soft Drop
- **q**: Quit

## Testing

Run the test suite:
```bash
mix test
```
